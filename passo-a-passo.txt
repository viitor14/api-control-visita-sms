ROTA DE USUARIO {
  CRIAR UM USUARIO.

  - SÓ CRIA USUARIO SE ESTIVER LOGADO. POREM JÁ VAI EXISTIR UM USUARIO PADRÃO CRIADO

  NA CRIAÇÃO DO USUARIO INFORMAR AS SEGUINTES INFORMAÇÕES: {

    * NOME *
    * EMAIL *
    * TELEFONE *
    * CARGO * (MASTER, GERENTE, RECEPÇÃO, SETOR)
    * SETOR * (PASSADO NO JSON COMO ID)
  }

  - PARA CRIAR USUARIO PARA RECEPÇÃO APENAS USUARIO MASTER
  - GERENTE SÓ CONSEGUE CRIAR USUARIO PARA O SETOR DELE
}

ROTA DE VISITANTE {
  USUARIO RECEPÇÃO CONSEGUE FAZER NOVO REGISTRO DE UM VISITANTE

  INFORMAÇÕES NECESSARIAS PARA CRIAÇÃO DE NOVO VISITANTE: {
    * NOME *
	  * CPF *
	  * TELEFONE *
  }

  - CONSEGUE VISUALIZAÇÃO DE TODOS VISITANTES CRIADOS
  - CONSEGUE VISUALIZAÇÃO DE UM VISITANTE ESPECIFICO ATRAVÉS DO CPF

}

-- APÓS PASSAR NA ROTA DE VISITANTE E FAZER A CRIAÇÃO DO CADASTRO DO MESMO DE SEGUIR PARA A ROTA DE VISITA ONDE VAI SER REGISTRADO A VISITA PARA QUAL SETOR A PESSOA VAI SE DIRIGIR --------

ROTA DE VISITA {

  USUARIO RECEPÇÃO E MASTER CONSEGUE TER A VISUALIZAÇÃO DE TODAS VISITAS PARA TODOS SETORES

  USUARIO RECEPÇÃO CONSEGUE CRIAR UMA NOVA VISITA INFORMANDO OS SEGUINTES DADOS: {
    * VISITANTE *
	  * SETOR *
	  * OBS * (MOTIVO DA VISITA)
  }

  USUARIO RECEPÇÃO CONSEGUE MUDAR O STATUS DA VISITA APENAS PARA FINALIZADO

  USUARIO GERENTE/SETOR SÓ CONSEGUE ALTERAR STATUS DA VISITA DO SEU SETOR

  USUARIO GERENTE/SETOR CONSEGUE ATUALIZAR O STATUS DA VISITA PARA AUTORIZADO/RECUSADO E PENDENTE

}
